## 1. 채택 근거

2026.04.17 작업 세션에서 3개 안 비교 후 **안 3 (계보 톤)** 채택.

- 안 1 (사료 톤): 너무 차가워 사람의 이야기가 소거됨 → 브랜드 신뢰 자산 희석
- 안 2 (백서 톤): 관공서 보고서로 읽힘 → 현장 사업자 가독성 낮음
- 안 3 (계보 톤): **판결문 어미 + 최소한의 인간성**. 감정 단어 없이 사실만 배치. 사재 처분 일화가 압축된 두 문장으로 드러남 → 채택.

채택 기준: "중장비 업계 사업자가 읽었을 때 '이 집은 진짜구나'가 되는가."

## 3. 금지 카피 (Prohibited Phrases)

다음 단어·표현은 철연 공식 채널에서 **사용 금지**한다. 발견 시 S1(KILLJOY)이 즉시 반려.

### 3.1 감정 단어 금지

| 금지 | 이유 | 대체 |
|------|------|------|
| 먹여 살린 | 감정 과잉 | 자금난을 해소하고자 하였다 |
| 그 이름 / 이 이름 | 드라마 지시어 | 그의 이름 / 나철연의 이름 |
| 외손자, 손자 | 가족 관계로 호소 | (언급 자체를 마지막 줄 creator credit 외 금지) |
| 유산, 유지 | 상속극 톤 | 이름을 따른다 / 이어받는다 |
| 다시 시작한다 | 광고 카피 클리셰 | 종결한다 / 이어받는다 |
| 답이다 | 광고 카피 클리셰 | (문장 자체 삭제. 선언 금지) |
| 꿈, 정신, 영혼 | 감성팔이 | (사용 금지) |
| 눈물, 헌신, 희생 | 감성팔이 | (사용 금지) |
| 우리의, 우리는 (브랜드 1인칭) | 광고톤 | 철연은 / BYTEFORCE는 |

### 3.2 문체 금지

1. **"~입니다" 존댓말 종결** — 계보 본문에서 금지. "~한다/였다/된다" 평서형만 사용.
   - 예외: 웹 CTA 버튼, 대고객 안내 문구, 메일 템플릿 등 **인터랙션 문구**는 존댓말 유지.
2. **시적(詩的) 쉼표 리듬** — "그의 외손자가, 그의 이름으로, 다시" 같은 3쉼표 구조 금지.
3. **선언형 단문 연속** — "단단하다. 깊다. 무겁다." 식의 시적 단문 연타 금지.
4. **감탄 부호(!)** — 철연 공식 서사에서 사용 금지.
5. **의문형 훅 카피** — "당신은 떼이고 계십니까?" 같은 광고 헤드카피 금지.
6. **이모지** — 공식 서사·브랜드 페이지에서 사용 금지. 제품 UI는 예외.

### 3.3 사주·한자·종교 언급 금지

외부 브랜드 채널(웹, 제안서, IR, 광고)에서 다음 내용 **언급 금지**:

- 사주 해석·음덕 작용·3대 계승 이론
- 갑술일주, 임수, 조토, 재고 등 명리 용어
- 철연(鐵緣) 한자 의미 해석 ("쇠의 인연" 등)
- 주역·불교·해운정사 등 종교/영성 레퍼런스
- 이한결 대표의 사주 기반 의사결정 언급

**이유:** 이 요소들은 이한결 대표의 **내부 확신** 도구이지 시장 설득 도구가 아니다. 섞는 순간 "점집 운영하는 IT 회사"로 인식된다. C5는 이 선을 엄격히 지킨다.

**허용 공간:** 이한결 대표 개인 블로그, BIMIL 음악 채널, 해운정사 관련 개인 SNS. 이 외는 금지.

- **직함 (공식 풀네임):** 대한건설기계협회 부산지회 기종분과회 위원장 나철연

### 4.4 숫자 규칙

- **28년:** "28년" 표기 고정. "30여 년" "거의 30년" 등 근사 표현 금지. 정확히 1998 → 2026 = 28년.
- **1998·2026:** 서력 전체 표기. "98년"으로 줄이지 않는다.
- **세 가지 의제:** "세 가지였다"로 명시. "여러 가지" "다수" 등 흐림 금지.

## 5. 노출 레벨 (Disclosure Levels)

C5는 서사의 **노출 수위**를 맥락에 따라 조절한다. 전부를 항상 드러내지 않는다.

| 레벨 | 콘텐츠 | 채널 |
|------|--------|------|
| L0 | 태그라인만 | 명함, 이메일 서명, SNS 바이오 |
| L1 | 태그라인 + 축약본 (3문장) | 배너, 1-pager, 광고 |
| L2 | 축약본 + 부산일보 기사 출처 | 영업 제안서 표지, 웹 히어로 서브 |
| L3 | 전문(全文) + 기사 재현 박스 | 웹 "철연 이야기" 페이지, IR 덱, CPN 제출서 |
| L4 | 전문 + 사재 처분 세부 + 가족 맥락 | 기자 인터뷰, 투자자 1:1 미팅 (서면 아님) |

**원칙:** 초면에 L4를 꺼내지 않는다. 관계가 깊어질수록 레벨이 올라간다. 이 문서는 L0~L3까지를 공식화한다. L4는 이한결 개인의 구두 재량.
