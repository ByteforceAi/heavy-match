- ✅ FAQ를 "인터랙션"으로 분류 → `~입니다` 유지
- ⚠️ FAQ를 "브랜드 서사"로 분류 → 평서형 전환

**권장:** FAQ는 본 PR에서 제외하고 브랜드 팀 결정 대기. 본 PR은 **narrative body 8건만 처리**.
