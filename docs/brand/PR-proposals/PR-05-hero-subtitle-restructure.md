## 판정 논리

§3.2.3 금지 예시 `단단하다. 깊다. 무겁다.`와 구조 비교:

| 항목 | 금지 예시 | 현재 서브 |
|------|----------|----------|
| 형태 | 형용사 단문 3개 | 숫자+명사 단문 3개 |
| 리듬 | 마침표 3연타 | 마침표 3연타 |
| 기능 | 시적 분위기 조성 | 제품 기능 나열 |
| 톤 | 감성 | 기술적·정량 |

**엄격 리딩:** 구조적으로 동일 → 금지 대상
**관대 리딩:** "숫자+명사"는 판결문·감정평가서 스타일에서도 발견됨 → 예외 가능

### 대체안 A · 쉼표 연결 (판결문 어미 유지)


**단점:** 길이 증가, 임팩트 약화 가능
